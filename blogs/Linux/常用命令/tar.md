# tar

## 常用命令
-A 新增压缩文件到已存在的压缩
-B 设置区块大小
-c 建立新的压缩文件
-d 记录文件的差别
-r 添加文件到已经压缩的文件
-u 添加改变了和现有的文件到已经存在的压缩文件
* **-x 从压缩的文件中提取文件**
* -t 显示压缩文件的内容
-z 支持gzip解压文件
-j 支持bzip2解压文件
* **-Z 支持compress解压文件**
* **-v 显示操作过程**

-l 文件系统边界设置

-k 保留原有文件不覆盖

-m 保留文件不被覆盖

-W 确认压缩文件的正确性

可选参数如下：

-b 设置区块数目

-C 切换到指定目录

* **-f 指定解压或者压缩的文件 必须放在命令选项参数的最后，后面接文件**
